
# 📊 Customer Segmentation & Revenue Analysis (RFM)

## 🚀 Project Overview
This project analyzes retail sales data to understand customer behavior using the RFM (Recency, Frequency, Monetary) model.

## 🛠 Tools Used
- Python
- Pandas

## 📊 Key Analysis
- Data Cleaning
- Revenue Analysis
- Pareto Analysis (Top Customers)
- RFM Analysis (Customer Behavior)
- Customer Segmentation

## 💡 Key Insights
- Electronics is the top revenue-generating category
- Female customers spend more
- Sales peak in May
- High-value customers are not frequent buyers

## 📈 Business Recommendations
- Focus on high spenders
- Improve customer retention
- Run campaigns in peak months
- Re-engage inactive customers

## 📂 Dataset
Retail Sales Dataset
